### Implementación de Singleton usando AOP
Proyecto desarrollado para demostrar la implementación del patrón de diseño Singleton mediante Programación Orientada a Aspectos (AOP) utilizando AspectJ. El aspecto intercepta las llamadas al constructor de la clase `ConfigManager` y garantiza que solo exista una única instancia durante toda la ejecución de la aplicación, sin necesidad de modificar el código original de la clase.

**Tecnologías utilizadas**
- Java
- AspectJ
- Programación Orientada a Aspectos (AOP)
- Visual Studio Code

**Enlace a Repositorio:**
[Ver proyecto](https://github.com/MAGSelPad/AOP_Singleton.git)


### Ranking Interactivo de Canciones
Aplicación desarrollada en JavaFX que presenta un ranking musical Top 10 mediante una interfaz gráfica interactiva. El sistema carga información de canciones desde archivos CSV, muestra su posición actual en el ranking y permite visualizar la evolución histórica de cada canción mediante animaciones e indicadores gráficos.

**Tecnologías utilizadas**
- Java
- JavaFX
- Programación Orientada a Objetos (POO)

**Enlace a Repositorio:**
[Ver proyecto](https://github.com/MiguelGalarza/proyecto-alertas)

## Tecnologías y Herramientas
* **Python:** Lenguaje de programación de alto nivel utilizado para automatización, análisis de datos, desarrollo de aplicaciones y resolución de problemas algorítmicos.
* **Java:** Lenguaje de programación orientado a objetos utilizado para desarrollar aplicaciones de escritorio, aplicaciones empresariales y proyectos académicos.
* **C:** Lenguaje de programación de propósito general utilizado para comprender estructuras de datos, gestión de memoria y fundamentos de programación de bajo nivel.
* **SQL:** Lenguaje utilizado para consultar, insertar, actualizar y administrar información almacenada en bases de datos relacionales.
* **HTML:** Lenguaje de marcado utilizado para estructurar el contenido de las páginas web. Ha sido la base para la creación de interfaces y sitios desarrollados durante el curso.
* **CSS:** Tecnología utilizada para dar estilo y mejorar la presentación visual de las aplicaciones web mediante colores, tipografías, animaciones y diseños responsivos.
* **JavaScript:** Lenguaje de programación utilizado para agregar interactividad a las páginas web, manipular el DOM y gestionar eventos del usuario.

* **Git:** Sistema de control de versiones utilizado para registrar cambios en los proyectos y facilitar el desarrollo colaborativo.
* **GitHub:** Plataforma para alojar repositorios, gestionar versiones de código y compartir proyectos con otros desarrolladores.
* **Docker:** Plataforma de contenedores utilizada para empaquetar aplicaciones junto con sus dependencias, facilitando su despliegue y ejecución en diferentes entornos.
* **Figma:** Herramienta de diseño colaborativo empleada para crear prototipos, interfaces de usuario y diagramas antes de implementar una aplicación.
* **VS Code:** Entorno de desarrollo ligero y extensible utilizado para programar, depurar y gestionar proyectos en distintos lenguajes y tecnologías.
* **Vite:** Herramienta de desarrollo moderna utilizada para crear proyectos web con una configuración rápida, recarga instantánea y optimización durante el desarrollo.
* **GitHub Pages:** Servicio de GitHub que permite publicar sitios web estáticos directamente desde un repositorio.
* **Vercel:** Plataforma utilizada para desplegar aplicaciones web de manera sencilla, permitiendo compartir proyectos en línea y automatizar procesos de publicación.
* **Firebase Realtime Database:** Servicio en la nube utilizado para almacenar y sincronizar datos en tiempo real entre múltiples usuarios y dispositivos.
* **Firebase SDK:** Conjunto de herramientas y bibliotecas que permiten integrar servicios de Firebase, como bases de datos, autenticación y almacenamiento, dentro de aplicaciones web.
